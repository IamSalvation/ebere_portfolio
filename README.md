## 📋 About This Project

This is a professional portfolio website for **Ebere Esther John** — a **Certified Virtual Assistant** and **Administrative Operations Specialist** helping founders and entrepreneurs reclaim 10+ hours weekly through reliable operational systems.

The site showcases her experience, certifications, services, client testimonials, and work samples with a premium **Navy Blue & Blonde** theme that reflects professionalism, trust, and reliability.

## 🚀 Features Breakdown

### 1. Hero Section
- Image on RIGHT side, content on LEFT (different from Bankesola's layout)
- **OCTAGON frame** for profile image (unique geometric shape)
- 3D tilt effect on profile image
- Animated stats counter (Years Experience, Connections, Projects Delivered)
- **Certified Virtual Assistant** badge
- Particle background effect

### 2. About Section
- **NO image** — clean text-only layout
- Professional bio with highlights
- 4 key value propositions

### 3. Services (6 Pillars)
- Inbox Management
- Calendar & Scheduling
- Data Entry & Reporting
- Document Preparation
- Client Communication
- Project Coordination
- *Horizontal scrolling carousel*

### 4. Experience
- 4 work experiences with skill tags
- Horizontal card-style layout
- Numbered timeline

### 5. Certifications
- Dedicated certifications section
- 4 credential cards
- Status badges (Completed / In Progress)

### 6. Work / Case Studies
- 6 project cards with modals
- Click to view detailed case study
- Includes problem, process, outcome, and tools

### 7. Work Samples (Auto-Moving Carousel)
- **Image-based cards** with overlay tags
- Auto-scrolling with pause on hover
- 8 project samples + 2 certificate cards
- Unique layout different from Bankesola's

### 8. Tools (Marquee)
- Infinite scrolling tools showcase
- Includes ChatGPT & Gemini
- Google Workspace, Notion, Trello, Zapier, Asana, and more

### 9. Testimonials (Auto-Sliding)
- 5 client testimonials
- Auto-sliding carousel
- Dots navigation

### 10. Booking Section
- Centered card layout
- Book Discovery Call (Google Calendar)
- Send Email
- Chat on WhatsApp
- Resume Download

### 11. Contact Section
- Social media links (LinkedIn, Instagram, X, YouTube, TikTok, GitHub, WhatsApp)
- Formspree contact form
